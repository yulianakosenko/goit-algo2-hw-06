## Завдання 2

Проведено порівняння точного підрахунку унікальних IP-адрес та алгоритму HyperLogLog.

Результати:

| Метод | Унікальні IP | Час виконання |
|-------|-------------:|--------------:|
| set | 28 | 0.001104 с |
| HyperLogLog | 28 | 0.059493 с |

### Висновок

Для невеликого набору даних структура set працює швидше.

Алгоритм HyperLogLog призначений для роботи з дуже великими наборами даних, де важливими є економія пам'яті та швидкість обробки. На невеликому файлі його переваги практично не проявляються.
